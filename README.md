# Focus-Guard AI BABYBOO
A Python tool that plays a video to shame you when you look away from your laptop (e.g., checking your phone).

## How to use:
1. Clone this repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Add alert media in the project root as either `1.mov` (video) or `1.png` (image).
4. Install VLC media player if using video alerts (`1.mov`): https://www.videolan.org/vlc/
5. Run: `python index.py`
